## Argentina Economic Changes Webhook
Technologies: `WebHooks`, `WebTask`, `IfThisThenThat`, `ServerLess Tech`, `Javascript`, `NodeJS`,` X-Ray`, `Cheerio` 

**Resume** A small script that runs as a [WebHook WebTask](https://webtask.io) that Parse changes in the Argentine Economy and notifies by Email Template

## Explanation / Steps
1. Used IFTTT to trigger the call to my WebTask using WebHooks
1. Then the WebTask runs using X-Ray and Cheerio to parse different webpages
1. Later, I format the parsed content as an Email Template
1. Finally trigger IFTTT to send the email
